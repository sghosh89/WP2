CHELSA data (version 2.1,https://chelsa-climate.org/timeseries/) folder, 
see detailed path here https://envicloud.wsl.ch/#/?prefix=chelsa%2Fchelsa_V2%2FGLOBAL%2F

3 TIF files need to be downloaded (from the paths given in 1981_2010/envidatS3paths.txt file and 
saved in 1981_2010/ folder) for the three variables described below.

shortname = bio5
longname = mean daily maximum air temperature of the warmest month
unit = °C
scale = 0.1
offset = -273.15
explanation = The highest temperature of any monthly daily mean maximum temperature

shortname = bio6
longname = mean daily minimum air temperature of the coldest month
unit = °C
scale = 0.1
offset = -273.15
explanation = The lowest temperature of any monthly daily mean maximum temperature

shortname = bio7
longname = annual range of air temperature
unit = °C
scale = 0.1
offset = 0
explanation = The difference between the Maximum Temperature of Warmest month and the Minimum Temperature of Coldest month
